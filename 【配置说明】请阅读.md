# Ninjemail - 多邮箱批量注册机 - 配置说明

## 平台用途
- **基础资源**: 邮箱是互联网服务的"身份证"
- **服务注册**: 用于注册各种平台和网站
- **验证码接收**: 接收各类验证码和通知
- **商业沟通**: 电子邮件营销和客户服务

## 功能特性
- **多平台支持**: 支持Gmail、Outlook、Yahoo等主流邮箱
- **批量注册**: 自动化批量创建邮箱账号
- **验证码处理**: 自动处理手机号验证
- **API接口**: 提供Python库调用接口

## 环境要求
- Python 3.8+
- Chrome/Chromium浏览器
- 手机号接码服务（SMS-Activate等）

## 使用方法

### 作为Python库使用

```python
from ninjemail import Ninjemail

# 初始化
ninja = Ninjemail()

# 创建Gmail账号
account = ninja.create_gmail(
    username="myusername",
    password="mypassword",
    phone_number="+1234567890"
)

# 创建Outlook账号
account = ninja.create_outlook(
    username="myusername",
    password="mypassword"
)
```

### 支持的邮箱平台
- ✅ Gmail (Google)
- ✅ Outlook/Hotmail (Microsoft)
- ✅ Yahoo Mail
- ✅ 更多平台持续添加中...

## 配置说明

1. **配置接码平台**:
   - 注册SMS-Activate或其他接码平台
   - 获取API Key
   - 在配置文件中设置

2. **配置代理（推荐）**:
   - 使用住宅IP代理
   - 避免IP被封

## 注意事项
- 邮箱注册需要手机号验证
- 建议使用优质接码平台
- Gmail注册最难，需要真实手机号
- Outlook相对容易
